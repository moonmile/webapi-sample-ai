# お寿司注文タブレット

# 概要

- テーブルに置いたタブレットで、お寿司を注文する
- 注文は Web API 経由で受け付ける
- 完了を知らせるために、サーバーから通知を受ける
- テーブル番号はあらかじめ設定済み

# 画面

## 着席画面

- 着席したときに人数を入力して「注文開始」ボタンを押す

## カテゴリ表示画面

- お寿司のカテゴリ一覧を表示する
- カテゴリからお寿司一覧を表示する
- お寿司の「カートに入れる」ボタンをクリックする

## カート画面

- カートに追加したお寿司の一覧を表示する
- 注文数を変更することができる
- 注文を確定するボタンを押す

## 注文履歴

- 注文したお寿司の一覧を表示する
- 注文のステータス（調理中、配達中、完了）

## 注文完了画面

- 注文が完了したことを知らせるメッセージを表示する
- 注文内容の確認ボタンを押すと、注文履歴画面に遷移する

## 精算画面

- 注文したお寿司の一覧を表示する
- 合計金額を表示する
- 支払い方法を選択する
- 支払いを確定するボタンを押す
